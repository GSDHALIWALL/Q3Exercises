HTML

HyperText Markup Languange

HTML stands for Hypertext Markup Languange. With the use of HTML, you are able to create websites. HTML was made by Tim Berners-Lee in 1993 and has been used to make websites. HTML makes it possible for information to be spread as websites are made.

CSS

Cascading Style Sheets

CSS or Cascading Style Sheets works with HTML. CSS can be used to customize on how the website looks. CSS also allows the seperation of style and content of your website.

Bootstrap

Bootstrap Version 4

Bootstrap is in websites, and allows for websites to adjust their size for specific devices. These devices include smartphones, laptops, tvs or any device that has a screen. Boostrap is easy to use and most tech giants use it.
